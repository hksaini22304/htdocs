About Me

Get to Know Me!

Hey, there! I'm Harjot Kaur Saini, a passionate creator specializing in web and mobile design. As a third-year student at Diablo Valley College, I am pursuing a degree in Interaction Design for Web and Mobile Platforms as well as Psychology, along with a Certificate in Business Marketing. My academic career is driven by a great passion about how design impacts human behavior, which I incorporate into every project I work on.

Through a range of courses, I've been able to work on projects that reflect real-world issues, allowing me to combine academic understanding with practical application. This diverse experience not only improves my technical abilities, but it also broadens my creative approach, ensuring that each design is both visually appealing and intuitively useful.

My Stack

Figma

I use Figma to create intuitive user interfaces, dynamic prototypes, and real-time collaborations with peers and mentors. The platform makes it easier to brainstorm, wireframe, and create high-fidelity prototypes, all while improving my UX/UI skills. It promotes iterative design, which improves my portfolio and academic work via professional-grade visual communication and continual development.

Framer

Framer helps me create dynamic, interactive prototypes that imitate user experiences. The platform supports motion design, interactive animations, and quick design iteration. I can see my ideas in real time and integrate complex micro-interactions and responsive layouts to improve my UX/UI design portfolio for academic and professional opportunities.

Notion

Notion organizes my academic work, research notes, and UX/UI design ideas. The platform organizes my class assignments, design materials, and project timetables. The configurable database and collaborative features ease task tracking and brainstorming, assuring efficiency while balancing learning and creativity.

HTML

I create websites, academic portfolios, and UX/UI design projects in HTML. Semantic tags and standardized markup help me organize and convey information in a clear way. HTML allows me to prototype concepts and try interactive layouts, which is the foundation of web design.

CSS

I integrate creative design ideas to academic assignments and personal projects by styling, visualizing web pages, and digital portfolios using CSS. I can regulate layout, typography, and color schemes to ensure interface consistency. CSS creates responsive, beautiful designs that follow latest UX/UI trends, improving user experience.

Creative Cloud

I use Adobe Creative Cloud for comprehensive design and creative media projects, leveraging industry-standard tools like Photoshop, Illustrator, and XD. This suite enables me to craft high-quality graphics, edit images, and produce compelling visual content for academic assignments and UX/UI prototypes. It enhances my creative workflow and portfolio development remarkably.

VS Code

VS Code is my preferred code editor for creating and managing online projects. It offers a clean and fast environment for developing HTML and CSS, as well as sophisticated extensions that boost productivity, ease debugging, and allow for seamless version control via Git. Whether I'm building responsive layouts or fine-tuning animations, VS Code keeps my process running smoothly and orderly.

Semrush

Semrush helps me assess website performance, digital marketing trends, and competitive landscapes. This tool informs content and user engagement methods for my UX/UI design projects. It optimizes design choices, improves search ranks, and aligns academic research with industrial norms using data.

Canva

Canva helps me develop attractive visuals, social media posts, academic projects and UX/UI portfolio presentations. My design process has been quicker by using its user-friendly interface and numerous templates, allowing creative expression and professional visual communication. Canva helps me deliver projects and improve as a designer.
